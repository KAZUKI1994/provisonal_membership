# 仕様
## 概要
1. メールアドレスを登録し、仮会員登録する。
2. 登録したメールアドレスに本会員登録URLを送信する
3. URLをクリックすると、本会員登録が開き本会員登録を行う。

## 詳細
1. メールアドレスを登録し、仮会員登録する。
    1. メールアドレス登録フォーム
    2. メールアドレス確認・送信
2. 登録したメールアドレスに本会員登録URLを送信する
3. URLをクリックすると、本会員登録が開き本会員登録を行う。
    1. 会員登録フォーム
    2. 登録確認
    3. 登録完了


## データベース
- 仮会員テーブル
- 本会員テーブル
